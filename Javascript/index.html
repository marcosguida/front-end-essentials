JavaScript

Introdução ao JavaScript

O JavaScript é uma linguagem de programação interpretada que permite adicionar interatividade e dinamismo às páginas da web. Com ele, você pode criar desde animações simples até jogos complexos e aplicações web robustas.

Tipos de Dados

  • Number: Números (ex: 10, 3.14)
  • String: Textos (ex: "Olá", "Mundo!")
  • Boolean: Verdadeiro ou falso (ex: true, false)
  • Object: Coleções de dados (ex: { nome: "João", idade: 30 })

Variáveis

Variáveis armazenam valores que podem ser usados posteriormente no código. Para declarar uma variável, utilize a palavra-chave `var`, `let` ou `const`, seguida do nome da variável e do valor inicial (opcional).

            var nome = "Pedro"; // Declaração com 'var'
            let idade = 25;   // Declaração com 'let'
            const pi = 3.14; // Declaração com 'const' (valor constante)

Características do JavaScript

  • Linguagem interpretada: O código é executado linha por linha, sem necessidade de compilação.
  • Orientada a objetos: Permite criar objetos reutilizáveis e modularizar o código.
  • Multiparadigma: Suporta programação funcional, imperativa e orientada a objetos.
  • Versátil: Aplicada em diversas áreas, desde desenvolvimento web até jogos e IoT.

Frameworks JavaScript

Frameworks facilitam o desenvolvimento web com JavaScript, fornecendo bibliotecas e ferramentas pré-construídas.

  • React: Cria interfaces web dinâmicas e componentizadas.
  • Angular: Framework completo para aplicações web robustas.
  • Vue.js: Leve e flexível, ideal para projetos de todos os tamanhos.

Sintaxe Básica

O JavaScript possui uma sintaxe relativamente simples e intuitiva, com base em instruções e blocos de código.

            // Comentário de uma linha
            /* Comentário de múltiplas linhas */

            // Função básica
            function soma(a, b) {
                return a + b;
            }

            // Chamada da função
            let resultado = soma(5, 3);
            console.log(resultado); // Imprime 8 no console
